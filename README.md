# Automated Question Generator  
An Automated Question Generator for English Reading Comprehension using T5, it takes user input text into wh questions and grammar questions.

## Installation

1. Install PyTorch from https://pytorch.org/get-started/locally/
2. Install requirements: `pip install -r requirements.txt`
3. Download nltk stopwords: `python -c "import nltk; nltk.download('stopwords')"`
4. Downlaod spacy model: `python -m spacy download en_core_web_sm`

## Run
Run the server: `python app.py`
