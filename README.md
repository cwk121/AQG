### Installation

1. Install PyTorch from https://pytorch.org/get-started/locally/
2. Install requirements using `pip install -r requirements.txt`
3. Download nltk stopwords using `python -c "import nltk; nltk.download('stopwords')"`
4. Downlaod spacy model using `python3 -m spacy download en_core_web_sm`

### Run

Run the server using `python app.py`
